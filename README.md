
# YouTube Mobile Redirector for Basilisk, Pale Moon, and Firefox

A lightweight browser extension that automatically redirects YouTube to its mobile version, providing a faster, simpler interface.


### Available for:
- **Firefox** → [https://addons.mozilla.org/es-ES/firefox/addon/youtube-mobile-redirector/](https://addons.mozilla.org/es-ES/firefox/addon/youtube-mobile-redirector/)
- **Basilisk** (2026.01.23) and **Pale Moon** → [Download XPI Extension](https://github.com/IamJony/youtubemobile-basilisk/releases/download/1.0/youtube-mobile_1.0_basilisk.xpi) *(manual install)*
